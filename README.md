# 🚀 Next CLI

**Next CLI** is a powerful project scaffolding tool built on top of the **Next.js** framework. It helps developers bootstrap full-featured web applications in seconds — saving hours of setup and configuration.

## ✨ Features

- 🔥 Blazing fast setup for Next.js projects
- 🧩 Modular configuration with support for:

  **Authentication**

  - `next-auth`
  - `clerk`
  - `kinde`
  - `none`

  **UI Libraries**

  - `shadcn-ui`
  - `radix-ui`
  - `chakra-ui`
  - `none`

  **ORMs**

  - `prisma`
  - `drizzle`
  - `none`

  **Databases**

  - `postgresql`
  - `mysql`
  - `sqlite`
  - `mongodb`
  - `none`

  **Emails**

  - `sendgrid`
  - `mailgun`
  - `postmark`
  - `resend`
  - `none`

  **Payments**

  - `stripe`
  - `lemon-squeezy`
  - `razorpay`
  - `paypal`
  - `none`

  **Analytics**

  - `vercel-analytics`
  - `google-analytics`
  - `none`

  **API Options**

  - `rest`
  - `trpc`
  - `graphql`
  - `hono`
  - `none`

- 📦 Package Manager Support:

  - `npm`, `pnpm`, `yarn`, `bun`
  - Executed via: `npx`, `pnpx`, `yarn`, `bunx`

- 💯 Developer-first interactive CLI experience

## 📦 Tech Stack

- **Base Framework:** Next.js
- **Customization Options:** Fully interactive selection of tools during setup

## 🛠️ Installation

Install the CLI globally using your preferred package manager:

```bash
npm install -g next-cli
# or
pnpm add -g next-cli
# or
yarn global add next-cli
# or
bun add -g next-cli
```
